# Examples

## Natural Numbers

This program prints the first n natural numbers

### Code

```
PRINT "Program to print natural numbers Written in !Python \n"

PRINT "Enter Limit \n"

INPUT a
LET i = 1

PRINT "Printing numbers from " i " to " a "\n"

WHILE i < a {
    PRINT i "\n"
    LET i = i + 1
}
```
